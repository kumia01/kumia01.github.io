Tags format the text

Tags are used to format the text. like the "heading 1" and "Paraghraph" that I have use here.

There are many basic tags and several more specialised tags. Browsers will gice default formating for these tags. Additionallu, the web developer may define how some or all these formating trags should be displayed. Also. the user may specify a custom theme (stylesheet) to be used, and this will override both default setting and tose set by the designer.

Basic tags

Headings -- witten h1, h2, ... h6 -- should be familiar from any word processing softaware. Similarly, paragraphs (p). Only important part: Remember to exit (close) any tag! That is the forwars-slash tag, as in /p.

There are several more formating tags, For example emphasis italics and strong/ bold/ fat text.

The most important principle, is consistency. You must use tags consistenly and ensure, for example, that you never follow a H1 with a H3 without a H2 inbetween. More in that later.

Tags defines text and layout

Then there are tags to make text clearly seperate from the rest. For example we use <code> </code> when we need a weapper for code/ script/ tags that should not be executed but displayed as on the webpage.

We have tags for ordered and unordered list (<ol> and <ul>).
